## 一、整体技术架构（清晰版）
```
前端三端 → 网关/请求 → Spring Cloud 微服务后端 → MySQL
                         ↓
                  Python LangChain 智能服务
                  (独立REST API提供智能能力)
```

### 技术栈明细
**前端**
- 前台管理端：Vue3 + Vite + Element Plus
- 管理员端：Vue3 + Vite + Element Plus
- 客户端：uni-app（微信小程序/APP/H5三端通用）

**后端**
- 框架：Spring Cloud Alibaba（微服务）
- 接口规范：RESTful API
- 安全：Spring Security + JWT
- 文档：Swagger/Knife4j

**数据库**
- MySQL 8.0
- Redis（缓存房间、订单、登录状态）

**智能模块**
- Python + FastAPI（提供REST接口）
- LangChain（智能问答、推荐、客服）
- 大模型：通义千问/文心一言/GLM（任选）

## 六、RESTful API 接口规范（统一标准）
**请求方式**
- GET 查询
- POST 新增
- PUT 修改
- DELETE 删除

**URL 示例**
```
GET    /api/room/list       查询房间
POST   /api/order/create    创建订单
PUT    /api/order/1/status  修改订单状态
DELETE /api/user/1          删除用户
```

**统一返回格式**
```json
{
  "code": 200,
  "msg": "成功",
  "data": {}
}
```
